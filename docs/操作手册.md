# 苏州市环境质量监测应用操作手册

## 应用简介

苏州市环境质量监测应用是一款专业的 environmental monitoring 移动应用，为苏州市民和环保工作者提供实时、准确的环境质量数据。应用支持查看苏州市当地实时/历史空气因子和水因子数据，GIS监测点位可视化，以及超标因子报警信息等功能。

## 主要功能模块

### 1. 空气质量监测

#### 1.1 实时空气质量数据
- **功能说明**：查看苏州市各监测点位的实时空气质量数据
- **操作步骤**：
  1. 进入应用首页，点击"空气质量"模块
  2. 选择"实时数据"标签页
  3. 在地图上选择监测点位，或从列表中选择
  4. 查看该点位的实时空气质量因子数据

#### 1.2 历史空气质量数据
- **功能说明**：查询指定时间段内的历史空气质量数据
- **操作步骤**：
  1. 进入"空气质量"模块
  2. 选择"历史数据"标签页
  3. 选择监测点位
  4. 选择查询时间范围（支持按小时、日、月查询）
  5. 查看历史数据趋势图表和详细数据列表

### 2. 水质监测

#### 2.1 实时水质数据
- **功能说明**：查看苏州市各监测点位的实时水质数据
- **操作步骤**：
  1. 进入应用首页，点击"水质监测"模块
  2. 选择"实时数据"标签页
  3. 在地图上选择监测点位，或从列表中选择
  4. 查看该点位的实时水质因子数据

#### 2.2 历史水质数据
- **功能说明**：查询指定时间段内的历史水质数据
- **操作步骤**：
  1. 进入"水质监测"模块
  2. 选择"历史数据"标签页
  3. 选择监测点位
  4. 选择查询时间范围
  5. 查看历史数据趋势图表和详细数据列表

### 3. GIS监测点位

#### 3.1 点位地图展示
- **功能说明**：在地图上可视化展示所有监测点位
- **操作步骤**：
  1. 进入应用首页，点击"监测点位"模块
  2. 地图自动加载并显示所有监测点位
  3. 点击点位图标查看该点位的基本信息和最新数据
  4. 支持地图缩放、平移等操作

#### 3.2 点位筛选
- **功能说明**：按类型、区域等条件筛选监测点位
- **操作步骤**：
  1. 在"监测点位"页面，点击筛选按钮
  2. 选择筛选条件：
     - 点位类型：空气质量监测点、水质监测点
     - 行政区域：姑苏区、工业园区、高新区等
     - 数据状态：正常、异常、离线
  3. 地图自动更新显示符合条件的点位

### 4. 超标报警

#### 4.1 实时报警信息
- **功能说明**：查看当前所有超标因子的报警信息
- **操作步骤**：
  1. 进入应用首页，点击"报警信息"模块
  2. 查看当前所有超标报警列表
  3. 点击报警条目查看详细信息：
     - 监测点位名称
     - 超标因子名称
     - 当前数值
     - 标准限值
     - 超标倍数
     - 报警时间

#### 4.2 报警历史记录
- **功能说明**：查询历史报警记录
- **操作步骤**：
  1. 在"报警信息"页面，选择"历史记录"标签页
  2. 选择查询时间范围
  3. 可按点位、因子类型筛选
  4. 查看历史报警记录列表

## 空气质量因子说明

### 主要监测因子

#### 1. PM2.5（细颗粒物）
- **全称**：细颗粒物，空气动力学直径小于等于2.5微米的颗粒物
- **来源**：工业排放、机动车尾气、建筑扬尘、燃煤等
- **健康影响**：可深入肺泡，影响呼吸系统和心血管系统
- **标准限值**（GB 3095-2012）：
  - 24小时平均：一级标准 ≤ 35 μg/m³，二级标准 ≤ 75 μg/m³
  - 年平均值：一级标准 ≤ 15 μg/m³，二级标准 ≤ 35 μg/m³
- **AQI分级**：
  - 优（0-50）：≤ 35 μg/m³
  - 良（51-100）：35-75 μg/m³
  - 轻度污染（101-150）：75-115 μg/m³
  - 中度污染（151-200）：115-150 μg/m³
  - 重度污染（201-300）：150-250 μg/m³
  - 严重污染（>300）：> 250 μg/m³

#### 2. PM10（可吸入颗粒物）
- **全称**：可吸入颗粒物，空气动力学直径小于等于10微米的颗粒物
- **来源**：道路扬尘、建筑工地、工业排放等
- **健康影响**：可进入呼吸道，引起呼吸系统疾病
- **标准限值**（GB 3095-2012）：
  - 24小时平均：一级标准 ≤ 50 μg/m³，二级标准 ≤ 150 μg/m³
  - 年平均值：一级标准 ≤ 40 μg/m³，二级标准 ≤ 70 μg/m³
- **AQI分级**：
  - 优（0-50）：≤ 50 μg/m³
  - 良（51-100）：50-150 μg/m³
  - 轻度污染（101-150）：150-250 μg/m³
  - 中度污染（151-200）：250-350 μg/m³
  - 重度污染（201-300）：350-420 μg/m³
  - 严重污染（>300）：> 420 μg/m³

#### 3. SO₂（二氧化硫）
- **全称**：二氧化硫
- **来源**：燃煤、工业排放、机动车尾气等
- **健康影响**：刺激呼吸道，引起支气管炎、哮喘等
- **标准限值**（GB 3095-2012）：
  - 1小时平均：一级标准 ≤ 150 μg/m³，二级标准 ≤ 500 μg/m³
  - 24小时平均：一级标准 ≤ 50 μg/m³，二级标准 ≤ 150 μg/m³
  - 年平均值：一级标准 ≤ 20 μg/m³，二级标准 ≤ 60 μg/m³
- **AQI分级**：
  - 优（0-50）：≤ 50 μg/m³
  - 良（51-100）：50-150 μg/m³
  - 轻度污染（101-150）：150-475 μg/m³
  - 中度污染（151-200）：475-800 μg/m³
  - 重度污染（201-300）：800-1600 μg/m³
  - 严重污染（>300）：> 1600 μg/m³

#### 4. NO₂（二氧化氮）
- **全称**：二氧化氮
- **来源**：机动车尾气、工业排放、燃煤等
- **健康影响**：刺激眼睛和呼吸道，加重呼吸系统疾病
- **标准限值**（GB 3095-2012）：
  - 1小时平均：一级标准 ≤ 200 μg/m³，二级标准 ≤ 200 μg/m³
  - 24小时平均：一级标准 ≤ 80 μg/m³，二级标准 ≤ 80 μg/m³
  - 年平均值：一级标准 ≤ 40 μg/m³，二级标准 ≤ 40 μg/m³
- **AQI分级**：
  - 优（0-50）：≤ 40 μg/m³
  - 良（51-100）：40-80 μg/m³
  - 轻度污染（101-150）：80-180 μg/m³
  - 中度污染（151-200）：180-280 μg/m³
  - 重度污染（201-300）：280-565 μg/m³
  - 严重污染（>300）：> 565 μg/m³

#### 5. CO（一氧化碳）
- **全称**：一氧化碳
- **来源**：机动车尾气、工业排放、燃煤等
- **健康影响**：与血红蛋白结合，影响氧气运输，引起头痛、恶心等
- **标准限值**（GB 3095-2012）：
  - 1小时平均：一级标准 ≤ 10 mg/m³，二级标准 ≤ 10 mg/m³
  - 24小时平均：一级标准 ≤ 4 mg/m³，二级标准 ≤ 4 mg/m³
- **AQI分级**：
  - 优（0-50）：≤ 2 mg/m³
  - 良（51-100）：2-4 mg/m³
  - 轻度污染（101-150）：4-14 mg/m³
  - 中度污染（151-200）：14-24 mg/m³
  - 重度污染（201-300）：24-36 mg/m³
  - 严重污染（>300）：> 36 mg/m³

#### 6. O₃（臭氧）
- **全称**：臭氧
- **来源**：光化学反应，由挥发性有机物（VOCs）和氮氧化物（NOx）在阳光照射下生成
- **健康影响**：刺激眼睛和呼吸道，引起咳嗽、胸闷等
- **标准限值**（GB 3095-2012）：
  - 1小时平均：一级标准 ≤ 160 μg/m³，二级标准 ≤ 200 μg/m³
  - 8小时平均：一级标准 ≤ 100 μg/m³，二级标准 ≤ 160 μg/m³
- **AQI分级**（8小时平均）：
  - 优（0-50）：≤ 100 μg/m³
  - 良（51-100）：100-160 μg/m³
  - 轻度污染（101-150）：160-215 μg/m³
  - 中度污染（151-200）：215-265 μg/m³
  - 重度污染（201-300）：265-800 μg/m³
  - 严重污染（>300）：> 800 μg/m³

#### 7. AQI（空气质量指数）
- **全称**：Air Quality Index，空气质量指数
- **计算方法**：根据PM2.5、PM10、SO₂、NO₂、CO、O₃等6项污染物的浓度计算
- **分级标准**：
  - 0-50：优（绿色）
  - 51-100：良（黄色）
  - 101-150：轻度污染（橙色）
  - 151-200：中度污染（红色）
  - 201-300：重度污染（紫色）
  - >300：严重污染（褐红色）

## 水质监测因子说明

### 主要监测因子

#### 1. pH值
- **全称**：氢离子浓度指数
- **说明**：表示水体的酸碱度
- **标准限值**（GB 3838-2002 地表水环境质量标准）：
  - Ⅰ类：6-9
  - Ⅱ类：6-9
  - Ⅲ类：6-9
  - Ⅳ类：5.5-9
  - Ⅴ类：5.5-9
- **超标影响**：pH值异常会影响水生生物生存，腐蚀管道

#### 2. 溶解氧（DO）
- **全称**：Dissolved Oxygen，溶解氧
- **说明**：水中溶解的氧气含量
- **标准限值**（GB 3838-2002）：
  - Ⅰ类：≥ 7.5 mg/L
  - Ⅱ类：≥ 6 mg/L
  - Ⅲ类：≥ 5 mg/L
  - Ⅳ类：≥ 3 mg/L
  - Ⅴ类：≥ 2 mg/L
- **超标影响**：溶解氧过低会导致水生生物缺氧死亡

#### 3. 化学需氧量（COD）
- **全称**：Chemical Oxygen Demand，化学需氧量
- **说明**：衡量水体中有机污染物含量的指标
- **标准限值**（GB 3838-2002）：
  - Ⅰ类：≤ 15 mg/L
  - Ⅱ类：≤ 15 mg/L
  - Ⅲ类：≤ 20 mg/L
  - Ⅳ类：≤ 30 mg/L
  - Ⅴ类：≤ 40 mg/L
- **超标影响**：COD过高表示水体有机污染严重，影响水质

#### 4. 氨氮（NH₃-N）
- **全称**：氨氮
- **说明**：水中以游离氨（NH₃）和铵离子（NH₄⁺）形式存在的氮
- **标准限值**（GB 3838-2002）：
  - Ⅰ类：≤ 0.15 mg/L
  - Ⅱ类：≤ 0.5 mg/L
  - Ⅲ类：≤ 1.0 mg/L
  - Ⅳ类：≤ 1.5 mg/L
  - Ⅴ类：≤ 2.0 mg/L
- **超标影响**：氨氮过高会导致水体富营养化，影响水生生态系统

#### 5. 总磷（TP）
- **全称**：Total Phosphorus，总磷
- **说明**：水中所有形态磷的总和
- **标准限值**（GB 3838-2002）：
  - Ⅰ类：≤ 0.02 mg/L（湖、库 ≤ 0.01 mg/L）
  - Ⅱ类：≤ 0.1 mg/L（湖、库 ≤ 0.025 mg/L）
  - Ⅲ类：≤ 0.2 mg/L（湖、库 ≤ 0.05 mg/L）
  - Ⅳ类：≤ 0.3 mg/L（湖、库 ≤ 0.1 mg/L）
  - Ⅴ类：≤ 0.4 mg/L（湖、库 ≤ 0.2 mg/L）
- **超标影响**：总磷过高会导致水体富营养化，引发藻类大量繁殖

#### 6. 总氮（TN）
- **全称**：Total Nitrogen，总氮
- **说明**：水中所有形态氮的总和
- **标准限值**（GB 3838-2002）：
  - Ⅰ类：≤ 0.2 mg/L
  - Ⅱ类：≤ 0.5 mg/L
  - Ⅲ类：≤ 1.0 mg/L
  - Ⅳ类：≤ 1.5 mg/L
  - Ⅴ类：≤ 2.0 mg/L
- **超标影响**：总氮过高会导致水体富营养化

#### 7. 高锰酸盐指数（CODMn）
- **全称**：高锰酸盐指数
- **说明**：用高锰酸钾作为氧化剂测定的化学需氧量
- **标准限值**（GB 3838-2002）：
  - Ⅰ类：≤ 2 mg/L
  - Ⅱ类：≤ 4 mg/L
  - Ⅲ类：≤ 6 mg/L
  - Ⅳ类：≤ 10 mg/L
  - Ⅴ类：≤ 15 mg/L
- **超标影响**：反映水体有机污染程度

#### 8. 五日生化需氧量（BOD₅）
- **全称**：Biochemical Oxygen Demand，五日生化需氧量
- **说明**：微生物分解水中有机物所需的氧气量
- **标准限值**（GB 3838-2002）：
  - Ⅰ类：≤ 3 mg/L
  - Ⅱ类：≤ 3 mg/L
  - Ⅲ类：≤ 4 mg/L
  - Ⅳ类：≤ 6 mg/L
  - Ⅴ类：≤ 10 mg/L
- **超标影响**：BOD₅过高表示水体有机污染严重

## 数据更新说明

### 更新频率
- **实时数据**：每5分钟更新一次
- **小时数据**：每小时整点更新
- **日均数据**：每日00:00更新前一日数据
- **月均数据**：每月1日00:00更新前一月数据

### 数据来源
- 数据来源于苏州市环境监测中心站
- 所有数据均经过严格的质量控制和审核
- 数据符合国家环境监测标准

## 常见问题

### Q1：如何理解AQI数值？
A：AQI数值越大，空气质量越差。建议：
- AQI ≤ 50：空气质量优，适合户外活动
- 51 ≤ AQI ≤ 100：空气质量良，一般人群可正常活动
- 101 ≤ AQI ≤ 150：轻度污染，敏感人群减少户外活动
- 151 ≤ AQI ≤ 200：中度污染，敏感人群避免户外活动
- AQI > 200：重度及以上污染，所有人群减少户外活动

### Q2：为什么不同监测点位的数值不同？
A：监测点位数值受多种因素影响：
- 地理位置：距离污染源的距离
- 气象条件：风向、风速、温度等
- 周边环境：交通流量、工业分布等
- 监测设备：设备精度和维护状态

### Q3：如何查看某个点位的历史数据？
A：操作步骤：
1. 进入"空气质量"或"水质监测"模块
2. 选择"历史数据"标签页
3. 选择监测点位
4. 选择查询时间范围
5. 查看数据趋势图表和详细列表

### Q4：报警信息如何理解？
A：报警信息包含：
- **监测点位**：发生超标的监测点位名称
- **超标因子**：超过标准限值的污染物名称
- **当前数值**：当前监测到的实际数值
- **标准限值**：该因子对应的标准限值
- **超标倍数**：当前数值相对于标准限值的倍数
- **报警时间**：首次检测到超标的时间

### Q5：数据更新不及时怎么办？
A：可能原因：
1. 网络连接问题：检查网络连接
2. 监测设备故障：联系相关部门
3. 数据审核中：数据需要审核后才能发布
4. 应用缓存：尝试刷新或重新加载数据
