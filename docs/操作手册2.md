## 账号注册与登录

### 1.1 用户注册

#### 功能说明
首次使用应用需要注册账号，注册后即可使用所有功能。

#### 操作步骤
1. 打开应用，点击"注册"按钮
2. 填写注册信息：
   - **用户名**：4-20个字符，支持字母、数字、下划线
   - **手机号**：11位中国大陆手机号码
   - **密码**：8-20个字符，需包含字母和数字
   - **确认密码**：再次输入密码进行确认
   - **验证码**：输入手机收到的6位数字验证码
3. 阅读并同意《用户协议》和《隐私政策》
4. 点击"注册"按钮完成注册

#### 注意事项
- 用户名一旦注册不可修改，请谨慎选择
- 手机号将用于接收验证码和重要通知，请确保手机号真实有效
- 密码请妥善保管，建议使用复杂密码提高安全性
- 验证码有效期为5分钟，过期需重新获取

### 1.2 用户登录

#### 功能说明
使用已注册的账号登录应用。

#### 操作步骤
1. 打开应用，进入登录页面
2. 输入登录信息：
   - **用户名或手机号**：输入注册时的用户名或手机号
   - **密码**：输入注册时设置的密码
3. 点击"登录"按钮

#### 登录方式
- **账号密码登录**：使用用户名/手机号 + 密码登录
- **手机验证码登录**：使用手机号 + 验证码登录（需先绑定手机）
- **记住密码**：勾选后下次打开应用自动填充密码
- **自动登录**：勾选后下次打开应用自动登录

#### 忘记密码
如果忘记密码，可以：
1. 点击登录页面的"忘记密码"
2. 输入注册时的手机号
3. 获取并输入验证码
4. 设置新密码
5. 使用新密码登录

## 个人信息管理

### 2.1 查看个人信息

#### 功能说明
查看当前登录账号的个人信息。

#### 操作步骤
1. 登录应用后，点击右下角"我的"标签
2. 进入"个人信息"页面
3. 查看以下信息：
   - 用户名
   - 手机号
   - 邮箱（如已绑定）
   - 注册时间
   - 账号状态

### 2.2 修改昵称

#### 功能说明
修改个人昵称，昵称将显示在个人资料中。

#### 操作步骤
1. 进入"我的" → "个人信息"
2. 点击"昵称"选项
3. 输入新昵称（2-20个字符）
4. 点击"保存"按钮

#### 注意事项
- 昵称不能包含敏感词汇
- 昵称修改后立即生效
- 昵称可以随时修改

### 2.3 修改头像

#### 功能说明
上传或更换个人头像。

#### 操作步骤
1. 进入"我的" → "个人信息"
2. 点击头像区域
3. 选择图片来源：
   - **拍照**：使用手机相机拍摄新照片
   - **从相册选择**：从手机相册选择已有照片
4. 裁剪和调整头像（可选）
5. 点击"确定"保存

#### 注意事项
- 支持 JPG、PNG 格式
- 图片大小不超过 5MB
- 建议使用正方形图片，尺寸建议 200x200 像素以上

### 2.4 绑定邮箱

#### 功能说明
绑定邮箱地址，用于接收重要通知和找回密码。

#### 操作步骤
1. 进入"我的" → "个人信息"
2. 点击"邮箱"选项
3. 输入邮箱地址
4. 点击"获取验证码"，查收邮箱验证码
5. 输入收到的验证码
6. 点击"确认绑定"

#### 注意事项
- 每个账号只能绑定一个邮箱
- 验证码有效期为10分钟
- 如需更换邮箱，需先解绑原邮箱

### 2.5 解绑邮箱

#### 功能说明
解除已绑定的邮箱。

#### 操作步骤
1. 进入"我的" → "个人信息"
2. 点击"邮箱"选项
3. 点击"解绑邮箱"
4. 确认解绑操作

#### 注意事项
- 解绑后无法通过邮箱找回密码
- 解绑后无法接收邮箱通知
- 解绑操作不可撤销

## 账号安全设置

### 3.1 绑定手机号

#### 功能说明
绑定手机号用于接收验证码、找回密码和重要通知。

#### 操作步骤
1. 进入"我的" → "账号安全"
2. 点击"手机号"选项
3. 如果未绑定手机号：
   - 输入手机号
   - 点击"获取验证码"
   - 输入收到的6位验证码
   - 点击"确认绑定"
4. 如果已绑定手机号，可以：
   - 查看当前绑定的手机号
   - 更换手机号（需先验证原手机号）

#### 更换手机号
1. 进入"账号安全" → "手机号"
2. 点击"更换手机号"
3. 验证原手机号：
   - 输入原手机号收到的验证码
4. 绑定新手机号：
   - 输入新手机号
   - 获取并输入新手机号验证码
   - 确认更换

#### 注意事项
- 每个手机号只能绑定一个账号
- 验证码有效期为5分钟
- 更换手机号后，原手机号将自动解绑

### 3.2 修改密码

#### 功能说明
修改登录密码，提高账号安全性。

#### 操作步骤
1. 进入"我的" → "账号安全"
2. 点击"修改密码"
3. 验证身份（三选一）：
   - **原密码验证**：输入当前密码
   - **手机验证码**：输入手机收到的验证码
   - **邮箱验证码**：输入邮箱收到的验证码（需已绑定邮箱）
4. 设置新密码：
   - 输入新密码（8-20个字符，需包含字母和数字）
   - 再次输入新密码确认
5. 点击"确认修改"

#### 密码要求
- 长度：8-20个字符
- 必须包含：至少一个字母和一个数字
- 建议包含：大小写字母、数字、特殊字符的组合
- 不能与用户名相同
- 不能使用过于简单的密码（如：12345678、password等）

#### 注意事项
- 修改密码后，所有已登录的设备将需要重新登录
- 请妥善保管新密码，避免泄露
- 建议定期更换密码，提高账号安全性

### 3.3 设置登录密码

#### 功能说明
如果注册时未设置密码，或使用手机验证码登录后，可以设置登录密码。

#### 操作步骤
1. 进入"我的" → "账号安全"
2. 点击"设置登录密码"
3. 输入新密码（需符合密码要求）
4. 再次输入密码确认
5. 点击"确认设置"

### 3.4 账号注销

#### 功能说明
永久删除账号及所有相关数据，注销后无法恢复。

#### 操作步骤
1. 进入"我的" → "账号安全"
2. 点击"账号注销"
3. 阅读注销须知和风险提示
4. 选择注销原因（可选）
5. 验证身份：
   - 输入登录密码，或
   - 输入手机验证码
6. 确认注销操作
7. 输入"确认注销"文字确认
8. 点击"确认注销"

#### 注销须知
- **数据删除**：注销后将永久删除以下数据：
  - 个人信息（用户名、手机号、邮箱等）
  - 历史查询记录
  - 收藏的监测点位
  - 报警订阅设置
  - 其他所有个人数据
- **不可恢复**：注销操作不可撤销，账号和数据无法恢复
- **注销条件**：注销前需确保：
  - 账号无未完成的业务
  - 无未处理的投诉或纠纷
- **注销后影响**：
  - 无法使用该账号登录
  - 无法找回账号数据
  - 手机号和邮箱可重新注册

#### 注意事项
- 注销前请备份重要数据
- 注销操作需要身份验证，确保是本人操作
- 注销后7天内，手机号和邮箱可重新注册新账号
- 如有疑问，请联系客服

### 3.5 登录设备管理

#### 功能说明
查看和管理已登录的设备，可以远程退出其他设备的登录。

#### 操作步骤
1. 进入"我的" → "账号安全"
2. 点击"登录设备管理"
3. 查看当前登录的设备列表：
   - 设备名称（如：iPhone 12、华为P40等）
   - 登录时间
   - 登录地点（IP地址）
   - 设备状态（当前设备/其他设备）
4. 管理设备：
   - **退出其他设备**：点击设备右侧的"退出登录"按钮
   - **退出所有设备**：点击"退出所有设备"按钮（当前设备除外）

#### 注意事项
- 退出其他设备后，该设备需要重新登录
- 建议定期检查登录设备，发现异常设备及时退出
- 如果发现未授权的设备登录，请立即修改密码

## 其他基础功能

### 4.1 消息通知设置

#### 功能说明
设置接收哪些类型的消息通知。

#### 操作步骤
1. 进入"我的" → "设置"
2. 点击"消息通知"
3. 开启或关闭以下通知类型：
   - **系统通知**：应用更新、维护公告等
   - **报警通知**：超标因子报警推送
   - **数据更新通知**：监测数据更新提醒
   - **活动通知**：活动、优惠等信息

#### 通知方式
- **应用内通知**：在应用内显示通知
- **推送通知**：手机系统推送（需开启应用通知权限）
- **短信通知**：发送短信（需绑定手机号）
- **邮件通知**：发送邮件（需绑定邮箱）

### 4.2 语言设置

#### 功能说明
切换应用显示语言。

#### 操作步骤
1. 进入"我的" → "设置"
2. 点击"语言设置"
3. 选择语言：
   - 简体中文
   - 繁体中文
   - English（如支持）

#### 注意事项
- 语言切换后立即生效
- 部分内容可能仍显示为原始语言

### 4.3 主题设置

#### 功能说明
切换应用主题（浅色/深色模式）。

#### 操作步骤
1. 进入"我的" → "设置"
2. 点击"主题设置"
3. 选择主题：
   - **浅色模式**：白色背景，适合白天使用
   - **深色模式**：黑色背景，适合夜间使用，节省电量
   - **跟随系统**：自动跟随手机系统主题

### 4.4 清除缓存

#### 功能说明
清除应用缓存数据，释放存储空间。

#### 操作步骤
1. 进入"我的" → "设置"
2. 点击"清除缓存"
3. 查看缓存大小
4. 点击"确认清除"

#### 注意事项
- 清除缓存不会删除个人数据
- 清除后首次加载可能较慢
- 建议定期清除缓存，保持应用流畅运行

### 4.5 检查更新

#### 功能说明
检查应用是否有新版本。

#### 操作步骤
1. 进入"我的" → "设置"

## 常见问题

### Q1：注册时收不到验证码怎么办？
A：可能原因和解决方法：
1. **手机号输入错误**：检查手机号是否正确
2. **网络问题**：检查网络连接，尝试切换网络
3. **验证码被拦截**：检查短信拦截设置，查看拦截记录
4. **发送频率限制**：同一手机号1分钟内只能发送1次，请稍后再试
5. **运营商问题**：联系运营商确认短信功能正常
6. 如仍无法解决，请联系客服：0512-12345

### Q2：忘记密码怎么办？
A：可以通过以下方式找回密码：
1. **手机验证码找回**：
   - 点击登录页"忘记密码"
   - 输入注册时的手机号
   - 获取并输入验证码
   - 设置新密码
2. **邮箱找回**（需已绑定邮箱）：
   - 点击"通过邮箱找回"
   - 输入绑定的邮箱
   - 查收邮件中的重置链接
   - 设置新密码
3. 如以上方式均无法找回，请联系客服协助处理

### Q3：如何更换绑定的手机号？
A：操作步骤：
1. 进入"我的" → "账号安全" → "手机号"
2. 点击"更换手机号"
3. 验证原手机号（输入验证码）
4. 输入新手机号并获取验证码
5. 输入新手机号验证码
6. 确认更换

### Q4：修改密码后无法登录？
A：可能原因：
1. **密码输入错误**：注意大小写，检查是否有空格
2. **新密码未保存成功**：重新修改密码
3. **账号被锁定**：多次输错密码可能导致账号临时锁定，等待30分钟后重试
4. 如仍无法解决，可通过手机验证码登录，然后重新设置密码

### Q5：账号注销后可以恢复吗？
A：不可以。账号注销是永久性的，注销后将：
- 永久删除所有个人数据
- 无法恢复账号
- 无法找回数据
- 手机号和邮箱可在7天后重新注册

因此，注销前请务必确认，并备份重要数据。

### Q6：如何退出登录？
A：有两种方式：
1. **退出当前设备**：
   - 进入"我的" → "设置"
   - 点击"退出登录"
   - 确认退出
2. **退出所有设备**：
   - 进入"我的" → "账号安全" → "登录设备管理"
   - 点击"退出所有设备"

### Q7：如何开启/关闭推送通知？
A：操作步骤：
1. **应用内设置**：
   - 进入"我的" → "设置" → "消息通知"
   - 开启或关闭相应通知类型
2. **系统设置**：
   - 进入手机"设置" → "应用管理" → "苏州市环境质量监测"
   - 开启"通知"权限

### Q8：如何清除应用数据？
A：有两种方式：
1. **清除缓存**（推荐）：
   - 进入"我的" → "设置" → "清除缓存"
   - 只清除缓存，保留个人数据
2. **清除全部数据**：
   - 进入手机"设置" → "应用管理" → "苏州市环境质量监测"
   - 点击"清除数据"
   - ⚠️ 注意：会删除所有数据，包括登录信息，需重新登录

### Q9：如何联系客服？
A：联系方式：
- **客服热线**：0512-12345（工作日 9:00-18:00）
- **在线客服**：应用内"我的" → "帮助与反馈" → "在线客服"
- **邮箱**：support@sz-env.gov.cn
- **意见反馈**：应用内"我的" → "设置" → "意见反馈"

### Q10：如何查看应用版本？
A：操作步骤：
1. 进入"我的" → "设置"
2. 点击"关于应用"
3. 查看版本号

## 安全提示

1. **密码安全**：
   - 使用复杂密码，定期更换
   - 不要将密码告诉他人
   - 不要在公共场所输入密码

2. **账号安全**：
   - 定期检查登录设备，发现异常及时退出
   - 不要使用他人设备登录
   - 注销账号前请备份重要数据

3. **隐私保护**：
   - 不要泄露个人信息
   - 谨慎授权第三方应用
   - 定期查看隐私设置

4. **防诈骗提示**：
   - 官方不会索要密码或验证码
   - 警惕虚假客服电话
   - 发现异常及时联系官方客服
