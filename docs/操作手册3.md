# App 修改密码通用指南

## 功能简介

本指南适用于平台内所有移动应用的密码修改功能。用户可以通过此功能定期更新账户密码，保障账户安全。

## 操作步骤

### 1. 进入个人中心

1. 打开应用，登录您的账户
2. 点击应用底部导航栏的"我的"或"个人中心"图标
3. 进入个人中心页面

### 2. 进入密码修改页面

1. 在个人中心页面，找到"账户设置"或"安全设置"选项
2. 点击进入设置页面
3. 选择"修改密码"或"更改密码"选项

### 3. 填写密码信息

1. 输入当前密码（用于验证身份）
2. 输入新密码（建议使用8位以上，包含字母、数字和特殊字符的组合）
3. 再次确认新密码（确保两次输入一致）

### 4. 提交修改

1. 确认所有信息填写正确
2. 点击"确认"或"提交"按钮
3. 系统将验证您的信息并完成密码修改

## 注意事项

### 密码安全建议

- **密码长度**：建议使用8位以上密码
- **密码复杂度**：建议包含大小写字母、数字和特殊字符
- **定期更换**：建议每3-6个月更换一次密码
- **避免重复**：不要使用与其他账户相同的密码

### 常见问题

#### Q1: 忘记当前密码怎么办？

A: 如果您忘记了当前密码，请使用"忘记密码"功能，通过手机验证码或邮箱验证来重置密码。

#### Q2: 修改密码后需要重新登录吗？

A: 是的，为了安全起见，修改密码成功后系统会自动退出登录，您需要使用新密码重新登录。

#### Q3: 新密码设置后多久生效？

A: 密码修改成功后立即生效，请妥善保管您的新密码。

#### Q4: 密码修改失败怎么办？

A: 请检查以下几点：
- 当前密码是否正确
- 新密码是否符合要求（长度、复杂度）
- 两次输入的新密码是否一致
- 网络连接是否正常

## 安全提示

1. **不要在公共场所修改密码**：避免在公共Wi-Fi环境下进行密码修改操作
2. **及时退出登录**：使用完毕后及时退出应用，特别是在公共设备上
3. **定期检查账户**：如发现异常登录记录，请立即修改密码并联系客服
4. **不要分享密码**：任何情况下都不要将密码告知他人，包括客服人员
